University Dissertation Artifact

PCG Dungeon Generator

Project Details

  • Platform: PC
  • Engine: Unity
  • Development Time: 6 Months
  • Role: Solo Developer
  • Tools: C#, Unity

Project Overview

For my university dissertation, I developed a PC-based procedural content generation (PCG) artifact focused on dungeon generation techniques commonly used in game development.

The project critically analyses multiple PCG algorithms, examining how each impacts spatial layout, dungeon readability, and player navigation.

Gameplay Mechanics

The system includes Cellular Automata, Binary Space Partitioning, Perlin Noise, Recursive Division, Drunkard Walk, and Recursive Backtracking.

Key Responsibilities & Contributions

  • Designed and implemented procedural generation algorithms
  • Conducted academic research into level design techniques
  • Built visual debugging tools
  • Created modular comparison systems
  • Refined dungeon navigation clarity
