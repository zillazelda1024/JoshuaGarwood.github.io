University Dissertation Artifact

PCG Dungeon Generator

Project Details

  • Platform: PC
  • Engine: Unity
  • Development Time: 6 Months
  • Role: Solo Developer
  • Tools: C#, Unity

Project Overview

For my university dissertation, I developed a PC-based procedural content generation (PCG) artifact focused on dungeon generation techniques commonly used in game development.

The project critically analyses how different algorithms impact dungeon layout, readability, and player navigation.

Gameplay Mechanics

The artifact implements six dungeon generation techniques: Cellular Automata, Binary Space Partitioning (BSP), Perlin Noise, Recursive Division, Drunkard Walk, and Recursive Backtracking.

Key Responsibilities & Contributions

  • Designed and implemented procedural dungeon generation algorithms
  • Conducted academic research into PCG techniques
  • Developed visual debugging tools for analysing dungeon structure
  • Created a modular system for algorithm comparison
  • Refined layouts to improve player navigation and clarity
