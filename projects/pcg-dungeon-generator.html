University Dissertation Artifact

PCG Dungeon Generator

Platform: PC

Engine: Unity

Development Time: 6 Months

Role: Solo Developer

Tools: C#, Unity

Project Overview

For my university dissertation, I developed a PC-based procedural content generation (PCG) artifact focused on dungeon generation techniques commonly used within game development. The project explores and critically analyses a range of PCG algorithms, examining how each method impacts spatial layout, dungeon readability, and player navigation.

The artifact functions as both a playable prototype and a research tool, allowing dungeon layouts to be generated and compared in real time. Emphasis was placed on balancing randomness with designer control to reflect real-world development practices.

Gameplay Mechanics

The project demonstrates six commonly used dungeon generation techniques: Cellular Automata, Binary Space Partitioning (BSP), Perlin Noise, Recursive Division, Drunkard Walk, and Recursive Backtracking.

Each algorithm can be individually selected and visualised, enabling direct comparison of dungeon density, room connectivity, corridor flow, and traversal patterns.

Key Responsibilities & Contributions

  • Designed and implemented multiple procedural dungeon generation algorithms
  • Conducted academic research into procedural level design techniques
  • Developed visual debugging tools for analysing dungeon structure
  • Created a modular system for comparing generation methods
  • Tested and refined dungeon layouts to improve navigation clarity
