University Dissertation Artifact

PCG Dungeon Generator

Project Details

  • Platform: PC
  • Engine: Unity
  • Development Time: 6 Months
  • Role: Solo Developer
  • Tools: C#, Unity

Project Overview

For my university dissertation, I developed a PC-based procedural content generation (PCG) artifact focused on dungeon generation techniques commonly used within game development.

The project explores and critically analyses a range of PCG algorithms, examining how each method impacts spatial layout, dungeon readability, and player navigation.

Gameplay Mechanics

The artifact functions as both a playable prototype and a research tool, allowing dungeon layouts to be generated and compared in real time.

Six techniques are implemented: Cellular Automata, Binary Space Partitioning (BSP), Perlin Noise, Recursive Division, Drunkard Walk, and Recursive Backtracking.

Key Responsibilities & Contributions

  • Designed and implemented multiple procedural generation algorithms
  • Conducted academic research into procedural level design
  • Developed visual debugging tools
  • Built a modular system for algorithm comparison
  • Refined layouts for improved navigation clarity
